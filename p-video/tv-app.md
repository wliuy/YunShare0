## 📖 配置教程
### 1. 云影空蒙（推荐）
由于该应用默认没有源，必须手动导入后方可使用：
* **导入路径**：`首页` -> `推送` -> `填入源名称与链接` -> `添加订阅源`

### 2. 天光云影
该应用默认内置了 GitHub (fanmingming) 的 IPv6 源。**注意：如果你的网络环境不支持 IPv6，默认源将无法播放，请按照以下步骤更换：**
* **导入路径**：`设置` -> `直播源` -> `自定义直播源` -> `添加其他直播源` -> `扫码/手动填入`
