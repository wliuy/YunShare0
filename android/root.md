# ⚡ Root 相关资源汇总
> 整理了 Root 后常用的权限管理工具、核心框架及各类实用功能模块。
---
**Root 权限是开启 Android 深度定制的钥匙，通过安装各类模块，可以实现功能增强、系统美化及去广告等深度操作。**
---
## 🛠️ 核心管理工具
玩机的第一步是获取并管理权限，这里收录了目前主流的 Root 方案。
* **Magisk (面具)**：最经典、兼容性最强的 Root 工具，拥有丰富的模块生态。
* **KernelSU / APatch**：新一代基于内核的 Root 方案，隐蔽性极高，更易通过各类 App 的 Root 检测。
## 🧩 常用核心模块
这些是 Root 之后几乎必装的基础框架与功能补丁。
* **LSPosed 框架**：玩机必备的核心框架，通过各种插件实现对系统和应用的深度定制。
* **Shamiko**：目前极受欢迎的隐藏 Root 模块，能够让各类敏感 App 正常运行。
* **Zygisk Next / BusyBox**：提供底层环境支持，是许多进阶模块运行的基础。
